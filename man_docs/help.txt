                               [ Available Flags ]

                ‘-p’ <NUM> display page at a set number of lines

                ‘-h’ displays help page.

                '-c' use w/ '-p', clears txt after displaying a chunk
                
                '-d' enables debug printouts for surface level debugging

                '-s' will search ALL directories, first one is displayed.
